# 🐮 Moo!ments  
### 소소한 행복을 기록하는 감정 기반 라이프 로그 앱  
*(Android & iOS 출시)*

> 일상 속에서 스쳐 지나가는 작은 행복을 기록하고,  
> 사용자가 자신의 감정 흐름을 인식하고 회고할 수 있도록 돕는 모바일 애플리케이션

## ✨ 주요 기능

### 📝 행복 순간 기록
- 짧은 텍스트로 빠른 기록
- 사진 첨부를 통한 순간 보존
- 감정 / 상황 태그 추가

### 🏷️ 태그 시스템
- 기본 제공 태그 + 사용자 정의 태그
- 태그별 기록 필터링
- 반복되는 행복 패턴 인식 가능

### 📅 캘린더 뷰
- 날짜별 기록 여부 한눈에 확인
- 태그가 표시된 시각적 회고 구조
- 감정 흐름을 시간 축으로 인지

### 📋 리스트 뷰
- 전체 기록 시간순 정렬
- 과거 기록 빠른 탐색
- 힘든 순간에 이전 행복을 다시 읽는 UX 제공

### 👤 마이 페이지
- 닉네임 기반 개인 공간
- 사용자 의견 수렴을 위한 소통 채널 연결
